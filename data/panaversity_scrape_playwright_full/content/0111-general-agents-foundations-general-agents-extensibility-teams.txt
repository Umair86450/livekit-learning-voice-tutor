Extensibility & Teams

This section covers the advanced patterns that make Claude Code a true development platform; hooks for automation, plugins that compose skills into workflows, the creator workflow for building and sharing extensions, and agent teams that coordinate multiple Claude instances on complex projects.

Lessons 15-25 take you from single-agent customization to multi-agent orchestration, parallel isolation with worktrees, remote session management, scheduled task automation, and event-driven channels that let agents react to file changes, git commits, and external triggers without polling.
